# uiua-playground

install uiua interpreter.

```shell
❯ cargo install uiua --version 0.13.0-rc.1
❯ ll ~/.cargo/bin/uiua
```

requires rustc and cargo to build.

```shell
❯ brew install rustup
❯ rustup toolchain install stable
❯ rustup default stable
❯ ls /opt/homebrew/opt/rustup/bin
```
